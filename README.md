# dotfiles_public (IN PROGRESS..)

Setup new Mac machine.

- Copy `Brewfile` to your directory
- Copy `setup.sh` to your directory
- Update `STRAP_GIT_NAME`, `STRAP_GIT_EMAIL`, and `STRAP_GITHUB_USER`
- Allow `setup.sh` to executable
- Run `./setup.sh`
